# Datawhale Hello-Agents 学习打卡仓库

这是一个围绕 Datawhale 社群任务整理的个人学习记录仓库，用来沉淀学习过程、实践结果、踩坑记录和阶段总结。

## 仓库目标

- 按照社区 `Task00 ~ Task06` 持续打卡
- 记录每个任务的学习输入、实践过程、问题与解决方案
- 保留个人理解，而不是简单搬运教程内容
- 让仓库既能服务自己复盘，也能作为公开作品集

## 推荐记录方式

每完成一个任务，优先补齐下面四类内容：

1. `任务要求`：今天需要完成什么，评审标准是什么
2. `实践过程`：看了哪些内容、跑了哪些代码、执行了哪些命令
3. `问题记录`：遇到什么报错，最后怎么解决
4. `学习心得`：用自己的话总结关键收获和下一步计划

## 提交建议

- 每个 Task 至少保留一次可追踪提交
- 截图建议放在对应任务目录下的 `截图/`
- 如果复现了代码，建议把个人改动写清楚，不直接覆盖原教程说明
- 不要提交 API Key、`.env`、本地虚拟环境等敏感或无关内容
